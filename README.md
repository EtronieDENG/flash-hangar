# 🚀 灵光格纳库 (Flash Hangar)

> **“把稍纵即逝的灵光一现，加工成永不被遗忘的思维记忆。”**

---

## 🏗️ 物理架构与厨房控温守则

格纳库根目录下严格限制物理层级，仅允许以下 5 个核心整备区：

* **`📁 00_Flash` （食材收集箱）**
    * **定义：** 从天而降的野生食材（脑中的灵光一现），一秒盲打，不假思索地以任何方式记录下来，错过了就没了。
    * **规范：** 纯粹的闪念盒，禁止建立任何子文件夹。
* **`📁 01_Projects` （炉子上的锅）**
    * **定义：** 正在执行的、有明确截止日期的短期项目（如：技术考证冲刺、美东14天旅游规划）。
    * **规范：** 严格控温，同时在锅里的项目**不大于 12 个**。项目结束直接撤火归档。
* **`📁 02_Areas` （冰箱冷藏室）**
    * **定义：** 长期保鲜、高频调用的常驻责任区（如：A 股红利资产追踪、院子景观改造）。
    * **规范：** 长期维护，无截止日期。你的深度长线原创思考直接原位写在这里的仪表盘中。
* **`📁 03_Resources` （常温食品柜）**
    * **定义：** 不碍事但需要时随时可用的主题资源、外部万能索引（如：AI Agent 脚本、网页剪藏、电子书路径、视频时间戳）。
    * **规范：** 仅作为客观资产索引。大资产（如大体积视频、安装包）**可借助 NAS 存储**，本地只留纯文本索引，保持仓库极致轻量。
* **`📁 04_Archives` （地下冷冻库 / 历史数据仓）**
    * **定义：** 已经宣告结束、彻底冻结的历史冷数据，零下十几度封存，仅备日后偶然复盘。

## 🛠️ ⚙️ Obsidian 逻辑层核心配置守则

### 1. 锁死“简短路径”链接

- **路径：** 设置 $\rightarrow$ `Files and links` $\rightarrow$ `Default link format`
- **配置：** 强行选择 **`Shortest path if possible`**
- **目的：** 链接只保留纯文件名。未来在 00 到 04 文件夹之间随意拖拽、搬运文件。
### 2. 开启双链自动更新

- **路径：** 设置 $\rightarrow$ `Files and links` $\rightarrow$ `Automatically update internal links`
- **配置：** **保持开启 (ON)**
- **目的：** 修改任何笔记文件名时，全库所有引用它的地方会自动同步修改，防止断联。
### 3. 新笔记默认砸进 `00_Flash`

- **路径：** 设置 $\rightarrow$ `Files and links` $\rightarrow$ `Default location for new notes`
- **配置：** 选择 `In the folder specified below` $\rightarrow$ 下方填入 **`00_Flash`**
- **目的：** 按下 `Ctrl + N` 自动盲打存入闪念盒，免去在记录瞬间纠结分类。

### 🥩 例子一：从天而降的“野生食材”（00_Flash） $\rightarrow$ 变成“长线私房菜”（02_Areas）

1. **随手一记（00_Flash）：**
    
    你在路上看新闻，看到大家都在追高科技半导体泡沫。你脑子里蹦出一个反向直觉：_“概念炒作博弈 10% 的利润风险太大，在老龄化周期里，反而是高分红的银行股、中证 A500 这种能稳定提供股息现金流的才是底层防御资产。”_
    
    - **操作：** 掏出手机，`Ctrl + N` 新建笔记自动生成在 `00_Flash` 里面，随手打字起名 `2026-07-02 银行红利直觉`，把这段话记下来。**2秒钟，野生食材摘下放着，不记回头就忘了。**
        
2. **空了去加工（00_Flash 原位修改）：**
    
    周末休息，你翻开这篇笔记，查了查资料，在里面补充因果闭环。
    
    - **操作：** 笔记继续留在 `00_Flash` 里，但在里面加了一行大白话：_“科技股炒的是概念利润，高分红银行股博弈的是确定性的股息。核心防御在于长线复利。”_ 这时，它就被你加工成了“原创思考卡片”。
        
3. **收进冰箱（02_Areas 调用）：**
    
    - **操作：** 这个投资逻辑长期有效，你打开你的冰箱长线资产大仪表盘 `📁 02_Areas / A股红利资产追踪.md`，直接在里面敲入双链：
        
        > “2026年仓位防守方向：参见我的原创红利因果推演 `[[2026-07-02 银行红利直觉]]`。”
        

### 🍳 例子二：偷学别人的“大厨配方”（03_Resources） $\rightarrow$ 放入常温食品柜（挂载 NAS）

1. **配方入库（03_Resources）：**
    
    你在网上看到一个上海本地大牛发的帖子，详细讲了“如何给院子做景观铺设”。
    
    - **操作：** 这是**别人的配方**。你把步骤复制下来，直接扔进 **`📁 03_Resources`（常温食品柜）**，起名叫 `[[大牛庭院景观铺设法]]`。
        
    - **挂载 NAS：** 顺便把大厨贴子里附带的 实操教学视频扔进 NAS 硬盘里，笔记里只写一行：_“视频存在 NAS 路径 XXX；重点在第 45 分钟看火山岩打底吸水动作。”_ 本地仓库只留纯文本，食品柜完全没有物理负担。
    
2. **冰箱高频盯着，照方抓药（02_Areas 真正调用）：**

- **操作：** 你平时需要长期、高频地折腾和维护自己那个院子。你点开冰箱冷藏室里长期保鲜的常驻责任区笔记：`📁 02_Areas / 院落景观改造进度.md`。在下周的实操计划里，你直接敲入双链：
    
    > “下周六计划：开始铺设左侧花坛石材以解决积水问题。具体的石头配比和底层火山岩吸水深度，**直接照抄食品柜里的配方：`[[大牛庭院景观铺设法]]` 第 45 分钟的操作。**”
        

### 🔥 例子三：炉子上的锅开了（01_Projects） $\rightarrow$ 撤火端进冷冻库（04_Archives）

1. **架起新锅（01_Projects）：**
    
    你需要规划十一国庆期间去美东（波士顿、纽约、华盛顿）的不开车、纯公共交通的安全旅行。这是一个**有截止日期的短期项目**。
    
    - **操作：** 确保当前的项目不超过 12 个。你在 `📁 01_Projects` 里新建文件夹叫 `📁 2026美东十一旅游规划`。里面建一篇主控笔记叫 `美东指挥部.md`。
        
2. **手忙脚乱时的复盘（产生短期思考）：**
    
    做攻略时，你发现了一个重大的安全盲区。
    
    - **操作：** 你立刻在 `美东指挥部.md` 里写下一行大白话复盘：_“纽约某些地铁站深夜极度不安全，绝对不能去。底层漏洞因果：以后行程规划，必须把‘交通选线’和‘安全避坑指南’的双链提前接进来。”_
        
3. **撤锅洗灶，端进冷冻库（04_Archives）：**
    
    十月中旬，你安全旅行归来。大晚宴这场战役彻底打完。
    
    - **操作：** 你冷酷地把整个 `📁 2026美东十一旅游规划` 文件夹，**物理拖拽扔进 `📁 04_Archives`（地下冷冻库）**。炉子空出来了，核心厨房重回极简。
        
    - **双链不死：** 虽然项目被冻结了，但因为你开启了 Obsidian 的“简短路径”，未来哪天你朋友要去美东旅游问你经验，你全局搜一下“纽约地铁”，当年在这口锅里写下的排雷复盘，依然能一秒捞出来。
